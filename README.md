# LOOM

Download instructions:

1. Go to https://randomcoffee.itch.io/loom
2. Download ZIP file
3. Unzip and click on the "Loom" folder
4. Click on Loom (APPLICATION)
5. Extract all files
6. Reopen Loom folder
7. Run Loom.exe
8. Enjoy!
